This code is thought for plotting geological structural data, such as faults bedding and kinematic markers, in stereonets easily modifiable using mplstereonet python package https://pypi.org/project/mplstereonet/.

I've had difficulty finding a program to plot accurately rakes from different type of input data (rake angles or trend/plunge data of kinematic indicators), so here it's possible to recalculate rake from trend/plunge and vice versa.

Classical stereonet showing great circle are accompanied by contour plots, rose diagrams (also adaptable to show paleocurrent) and gaussian polymodal fit of data distributions, to analyze different families of structural data.

The code here furnished was written in Colab exploiting built-in Gemini LLM mostly to uniform data management through the code (I'm quite new to Pandas...)

It's my very first time doing this, please contact me for every type of advise.

Some plots produced by earlier versions of the code can be found here: https://doi.org/10.1029/2024GC012005
